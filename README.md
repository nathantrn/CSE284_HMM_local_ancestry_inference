# CSE284_HMM_local_ancestry_inference

This repository implements an end-to-end Hidden Markov Model (HMM) approach for local ancestry inference on phased haplotypes, replicating the basic functionality of FLARE. The model treats ancestry along a chromosome as a sequence of hidden states (AFR/EUR/EAS), with transitions driven by recombination distance and emissions based on ancestry-specific allele frequencies estimated from a reference panel.
